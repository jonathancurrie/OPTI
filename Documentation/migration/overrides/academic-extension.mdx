Once you have completed the form below, you will be able to download the OPTI Toolbox Academic Extension. The extension currently consists of the [SCIP](../solvers/scip.md) 64-bit Windows MEX file, `scip.mexw64`.

After downloading the extension, place the MEX file in:

```text
OPTI_Install_Location/Solvers/
```
