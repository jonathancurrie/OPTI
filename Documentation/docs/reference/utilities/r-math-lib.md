## RMathLib [Supplied]
<small></small>
RMathLib is an OPTI interface to the R standalone Math Library. The library has been compiled for Windows and a parallelized and vectorized MEX interface written to enable high-speed statistical functions.

### Included Functions
To see what functions are included as part of the RMathLib library, type the following at the MATLAB command line:

` >> rmathlib`
