## AMPL Solver Library [Supplied]
<small></small>
ASL is an interface which can read AMPL .NL files. It includes callbacks for nonlinear functions, allowing most AMPL models to be read into MATLAB. The interface automatically detects the problem type (LP, MILP, QP, MIQP, NLP, MINLP) and builds the required OPTI structure.
