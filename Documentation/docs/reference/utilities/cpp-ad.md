## CppAD [Supplied]
<small></small>
CppAD (C++ Automatic Differentiation) is the de-facto open-source standard in overloaded automatic differentiation in C++, and has been included in OPTI from v2.10 when generating models from the SymBuilder object.

### Use in OPTI
Note CppAD is not used to differentiate MATLAB models, only C++ models that have been auto-coded using SymBuilder. See the [SymBuilder Page](../../guides/advanced/sym-builder.md) for an example on how to use CppAD with OPTI.
