Complete the verification and confirm that the download will only be used for academic purposes to access the OPTI Toolbox Academic Extension. The extension currently consists of the [SCIP](../solvers/scip.md) 64-bit Windows MEX file, `scip.mexw64`.

After downloading the extension, place the MEX file in:

```text
OPTI_Install_Location/Solvers/
```
