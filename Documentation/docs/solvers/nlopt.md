## NLOPT [Supplied]
<small></small>
NLOPT is a collection of over 20 nonlinear program solvers, including solvers for both global and local problems. It also contains a number of solvers for derivative free optimization. While most algorithms are limited to bound constraints, some allow linear / nonlinear constraints.
