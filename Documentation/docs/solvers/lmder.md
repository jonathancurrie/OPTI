## MINPACK Levenberg Marquardt [Supplied]
<small></small>
LM<sub>DER is a MINPACK routine for solving NLS problems. It is also compiled to include LM</sub>DIF (internal finite differences).

Note LM_DER only solves unconstrained problems.
