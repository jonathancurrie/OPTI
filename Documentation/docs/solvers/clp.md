## Coin-OR Linear Programming [Supplied]
<small></small>
CLP solves LPs using a Primal Simplex method, and QPs using a Dual Simplex method. In addition, it contains a barrier solver which can be used to solve both problem types. It is one of the best open-source LP solvers available.
