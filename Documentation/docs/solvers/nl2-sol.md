## Adaptive Nonlinear Least Squares [Supplied]
<small></small>
NL2SOL solves NLS problems using an adaptive model algorithm. It is also compiled to include NL2SNO (internal finite differences). Probably the best NLS solver I have come across!

Note NL2SOL only solves unconstrained and bounded problems.
