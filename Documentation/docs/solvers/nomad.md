## NOMAD [Supplied]
<small></small>
NOMAD uses a Mesh Adaptive Direct Search algorithm to solve non-differentiable and global nonlinear programs. It also contains a facility to solve non-convex MINLPs, and does this quite well for problems up to a few hundred variables. Excellent solver for derivative free optimization.
