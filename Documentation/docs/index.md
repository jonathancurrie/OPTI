If you are interested in optimization, use MATLAB and like free stuff, OPTI could be for you. See the [What Is OPTI](./overview.md) section for details on solving linear, nonlinear, continuous and discrete optimization problems using MATLAB!

To get right into it, jump to the [downloads](./project/download.md) page!
